## 🧠 Overview

ToneShift is a production-ready full-stack AI application that transforms text tone using secure REST APIs and modern frontend architecture.

The platform implements:

- Stateless JWT authentication
- Spring Security authorization
- PostgreSQL database integration
- OpenAPI (Swagger) documentation
- Cloud deployment on Render & Vercel
- Environment-based configuration handling

This project demonstrates real-world backend architecture, secure API design, and cloud deployment experience.
